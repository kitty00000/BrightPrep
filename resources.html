Resources

How Does This Work? We offer a variety of plans and schedules tailored to different learning needs. You can open them in your drive, make a copy, and edit as needed. Each comes with a detailed description to guide you!

Weekly Study Timetable

This is a tool you can use if you like to plan your day using what you should accomplish for each subject you have. At the bottom of the table for the week, there is a section where you can write any additional notes.

Weekly Subject Planner

This is a tool that you can use if you like to plan out your day and week based on specific time slots. There is a table beside it for listing your subjects and a to-do list where you can add tasks.
